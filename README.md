Memory Box is a web-based journaling platform where users can store and organize personal memories in interactive drawers. It also includes a “Rant  Box” for expressing thoughts or frustrations. Built using HTML, CSS, JavaScript, and basic PHP.
📦 Memory Box – Online Memory Journal

Memory Box is a simple web-based journaling platform where users can store and organize their personal memories in a creative way.

Instead of a traditional diary, the website provides interactive memory drawers where different memories can be written and saved. Each drawer represents a category of memories, allowing users to organize their thoughts and experiences.

The website also includes a Rant Box, a special space where users can freely vent their frustrations, thoughts, or complaints without judgment.

This project was created as a learning project to practice frontend and basic backend web development.
✨ Features

🗂 Memory Drawers
Different drawers allow users to store memories in separate sections.

✍️ Online Journaling
Users can write personal memories and reflections.

😤 Bitching Box
A dedicated space to express frustrations or random thoughts.

🎨 Interactive User Interface
Drawer-style design that makes journaling more engaging.

⚙️ Basic Backend Handling
PHP is used for simple backend processing and form handling.
